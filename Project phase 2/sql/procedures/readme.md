# tudyBuddy SQL Stored Procedures

**Overview**

This folder contains all SQL stored procedures developed for the StudyBuddy Study Management and Gamification System. These procedures support essential backend operations across the platform, including task handling, topic management, timer session processing, user statistics computation, leaderboard updates, challenge tracking, and other automated database behaviors.

Each procedure is designed to follow the optimized schema and indexing strategy used throughout the database, helping ensure fast, efficient performance as the dataset grows. This folder serves as the centralized location for all procedural logic created by the team, keeping the project organized and maintainable.
